# Fand Server Website

Fand Server 官网，基于 Astro 和 Tailwind CSS 构建。

## 开发

安装依赖：

```bash
bun install
```

启动开发服务器：

```bash
bun run dev
```

生产构建：

```bash
bun run build
```

## 目录

```text
src/pages       Astro 页面路由
src/components  Astro 组件
src/layouts     共享布局
src/styles      全局样式与 Tailwind 主题
src/content     新闻与项目更新
public          静态资源
```
